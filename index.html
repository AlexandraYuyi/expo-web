¿Qué es una transición?

Sin transición

                        .caja {
                            background-color: cornflowerblue;
                        }

                        .caja:hover {
                            background-color: orange;
                        }

Con transición

                        .caja {
                            transition: background-color 500ms linear;
                        }

Agregar un delay

Inmediata

                        .nodelay {
                            transition-delay: 0s;
                        }

Usando un delay

                        .delay {
                            width: 100px;
                            height: 100px;
                            background: red;
                            transition: width 3s;
                            transition-delay: 1s;
                        }
                        
                        .delay:hover {
                            width: 300px;
                        }

¿Qué son Animaciones?

Transition simple

                        .noanimada {
                            width: 100px;
                            height: 100px;
                            background-color: red;
                        
                            transition: background-color 3s linear;
                        }
                        
                        .noanimada:hover {
                            background-color: green;
                        }

Cambios intermedios con animación

                        @keyframes rainbow {
                            0%   {background-color:red;}
                            10%   {height: 120px}
                            25%  {background-color:yellow; height: 140px}
                            50%  {background-color:blue; height: 160px}
                            75%  {background-color:green; height: 180px}
                            100% {background-color:purple; height: 200px}
                        }
                        
                          .animada:hover {
                            animation-name: rainbow;
                            animation-duration: 3s;
                          }

Animación con forwards y delay

                        .animada.animada-forwards:hover {
                            animation-fill-mode: forwards;
                            animation-delay: 1s;
                        }

Transiciones con Javascript

Evento transition event listener

                    const button = document.querySelector('.button');

                    function toggleClass() {
                        button.classList.toggle('active');
                    }

                    function addClass() {
                        button.classList.add('finished');
                    }

                    button.addEventListener('click', toggleClass);
                    button.addEventListener('transition', toggleClass);
                    button.addEventListener('transition', addClass);
