Transiciones

en la web

¿Qué es una transición?

Sin transición

                        .caja {
                            background-color: cornflowerblue;
                        }

                        .caja:hover {
                            background-color: orange;
                        }

Con transición

                        .caja {
                            transition: background-color 500ms linear;
                        }

Agregar un delay

Inmediata

                .nodelay {
                    transition-delay: 0s;
                }

Usando un delay

                    .delay {
                        width: 100px;
                        height: 100px;
                        background: red;
                        transition: width 3s;
                        transition-delay: 1s;
                    }
                    
                    .delay:hover {
                        width: 300px;
                    }

Tipos de funciones de intervalo de Transición

Transicion de tipo: linear

                    #div1 {
                        transition-timing-function: linear;
                    }

Transicion de tipo: ease

                    #div2 {
                        transition-timing-function: ease;
                    }

Transicion de tipo: ease-in

                    #div3 {
                        transition-timing-function: ease-in;
                    }

Transicion de tipo: ease-out

                    #div4 {
                        transition-timing-function: ease-out;
                    }

Transicion de tipo: ease-in-out

                    #div5 {
                        transition-timing-function: ease-in-out;
                    }

¿Qué son Animaciones?

Con animación

                        @keyframes example {
                            0%   {background-color: red;}
                            25%  {background-color: yellow;}
                            50%  {background-color: blue;}
                            100% {background-color: green;}
                          }
                        
                          .animada{
                            width: 100px;
                            height: 100px;
                            background-color: red;
                          }
                        
                          .animada:hover {
                            animation-name: example;
                            animation-duration: 4s;
                          }
